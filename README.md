# Finnovu (MVP)

A cross-platform Expense Tracker app built with Flutter.

## Features
- Manual Expense Entry
- Local Storage (SQLite)
- Dashboard & Analytics
- Multi-platform support (Android, iOS, Web, Windows, Linux, macOS)

## Getting Started
1. `flutter pub get`
2. `flutter run`
